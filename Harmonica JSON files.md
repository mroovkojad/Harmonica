JSON files describing harmonica contain following info:

1. Tuning (e.g. Richter), see Tunings
1. Number of holes - # of columns in diagram
1. Rows - # of rows in diagram: each
    * including row for hole numbers
    * one row for each *articulation* used (see below)
    * rows may be *shared* between articulations e.g **blow bend** and **overblow**
1. Articulation:  technigue used to produce sound:
    * blow or draw
    * blow/draw bend (includning level)
    * overblow or overdraw
    * valved bend
